# molozi.ly

Site para colocar links na biografia do Instagram, feito usando HTML, CSS, JS e Python Flask

-------------------------------------------------------------
- git clone https://github.com/jeanfiorentin/molozi.ly.git

- cd molozi

- pip install -r requeriments.txt

- run flask
